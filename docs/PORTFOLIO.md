# Portfolio Case Study

## Project Name

EvidenceIQ: Agentic AI Investigation Command Center

## Problem

Investigators often work across messy emails, notes, transactions, and reports.
The hard part is not just search; it is turning scattered evidence into cited
facts, entity relationships, timelines, and defensible next steps.

## Solution

EvidenceIQ provides a local-first investigation workspace:

- Evidence Vault for uploads and source tracking.
- Entity Graph for relationship discovery.
- Timeline Builder for chronological reconstruction.
- Risk Console for explainable investigation leads.
- Investigator Q&A with citation-backed answers.
- Memo Generator for professional case summaries.

## Technical Highlights

- Standard-library core engine for zero-budget portability.
- Deterministic retrieval and refusal behavior.
- SQLite persistence for local case storage.
- Rule-based entity extraction with clear extension points for spaCy/LLMs.
- Graph traversal for relationship paths.
- Citation-first answer and memo generation.

## Resume Bullet

Built EvidenceIQ, a zero-cost AI investigation command center that transforms
raw evidence into searchable intelligence, relationship graphs, timelines,
risk signals, and citation-backed memos using local NLP, retrieval, graph
analytics, SQLite, and tool-using agent workflows.
